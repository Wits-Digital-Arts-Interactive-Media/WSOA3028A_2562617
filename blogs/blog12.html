Strengths and Weaknesses

What have I done really well from my goals?

Since its the beginning of the course, it would be good to set some goals for things that I want to achieve by the end. Below are my goals for this semester:

  • My first goal for this course is to understand, and learn how to code in CSS and JavaScript. I learnt HTML in CAT class when I was in high school so I still remember the basics. However, I still plan on putting in the same amount of effort when learning/re-learning all of them.
  • My second goal is to create a functional, aesthetically pleasing website that reflects my art style, abilities and the effort that I put into my work.
  • My last goal is to achieve a mark that is 65% and above. This is the standard that I put for all my courses and it is how I know that I have done well. Getting a distinction would also be great, but I don't want to put that kind of pressure on myself on just the first website.
